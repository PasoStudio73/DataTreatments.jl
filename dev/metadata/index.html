Metadata

The metadata module defines the core types used to describe individual columns (features) inside a DataTreatment. Each struct captures what kind of data a column contains together with validity information (which rows are valid, missing, or NaN).

There are four concrete subtypes of AbstractDataFeature:

TypeUse case
DiscreteFeatCategorical / discrete columns (e.g., labels, classes)
ContinuousFeatScalar numeric columns (e.g., measurements, sensor readings)
AggregateFeatMultidimensional columns flattened via windowed aggregation
ReduceFeatMultidimensional columns whose size is reduced while preserving dimensionality
