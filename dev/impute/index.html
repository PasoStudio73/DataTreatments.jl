Imputation

Wrappers around Impute.jl for handling missing and NaN values in datasets. Imputors can be passed to load_dataset via a TreatmentGroup.

Supported Imputors

The following imputors are re-exported from Impute.jl and can be used directly:

ImputorDescription
InterpolateLinear interpolation between valid values
LOCFLast Observation Carried Forward
NOCBNext Observation Carried Backward
SubstituteReplace with a summary statistic (e.g. mean)
SVDSVD-based matrix completion

Imputors can be chained in a tuple and are applied in order:

# fill gaps by interpolation, then carry forward/backward
# any remaining leading/trailing missings
impute = (Interpolate(), LOCF(), NOCB())

Dispatch Behaviour

_impute dispatches on the type of the input data:

  • AbstractMatrix: imputors are applied column-wise (dims=2).
  • AbstractMatrix{T} where T<:Union{Missing,Float,AbstractArray{<:Float}}: imputors are applied without a dimension constraint.
  • Scalar or array-valued element: imputors are applied only if the element is a non-missing AbstractArray; scalars are returned unchanged.

In all cases NaN and "NULL" are first declared as missing, and disallowmissing is called if no missing values remain after imputation.

Internal wrapper around Impute.jl that handles missing and NaN value imputation. Dispatches on the type of data:

  • AbstractArray: applies each imputor column-wise (dims=2).
  • AbstractMatrix{T} where T<:Union{Missing,Float,AbstractArray}: applies each imputor without specifying a dimension.
  • Scalar or array-valued float element: applies imputors only if data is a non-missing AbstractArray, otherwise returns unchanged.

In all cases, NaN and "NULL" are first declared as missing via Impute.declaremissings, then each imputor in the tuple is applied in order via Impute.impute!. If no missing values remain after imputation, disallowmissing is called to tighten the element type.

Arguments

  • data: Input data to impute. Can be an AbstractArray, an AbstractMatrix of floats, or a single scalar/array-valued float element.
  • impute::Tuple{Vararg{<:Impute.Imputor}}: Ordered tuple of imputors (e.g. (Impute.Interpolate(), Impute.Fill(0.0))).

Returns

  • The imputed data, with missing values removed from the type if possible.
